# Singular Value Decomposition and Image Compression

A MATLAB project demonstrating Singular Value Decomposition (SVD) for image compression and noise filtering.

**MTH 343 - Applied Linear Algebra - Fall 2025**

**Group Members**:

- Anthony Vu
- Jacob Purvis

## Overview

This project explores the practical applications of SVD in linear algebra through:

- Visualization of 2D matrix transformations using SVD decomposition
- Image compression using truncated SVD
- Noise filtering in images

## Files

- `lab14.m` - Main MATLAB script containing all demonstrations
- `einstein.jpg` - Sample image for compression testing
- `checkers.pgm` - Sample image for noise filtering demonstration

## Requirements

- MATLAB with Image Processing capabilities
- Images: `einstein.jpg` and `checkers.pgm`
